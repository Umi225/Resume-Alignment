## 设计系统

### 色彩
- **主色**：`#1a1a1a`（正文黑）
- **次要**：`#666666`（次要文字）
- **辅助**：`#999999`（辅助文字 / 占位符）
- **边框**：`#e5e5e5`（分割线）
- **背景**：`#ffffff`（主背景）、`#f5f5f5`（次级背景）
- **强调**：`#2563eb`（蓝色，仅用于交互状态，保持克制）
- **成功/采纳**：`#16a34a`（AI 建议已采纳标识）

### 字体
- 系统字体栈：`"PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif`
- 简历渲染使用同等栈，确保打印时不依赖外部字体加载

### 间距
- 基础单位：4px
- 区块间距：24px ~ 32px
- 卡片内边距：16px ~ 24px

## 页面结构

### 1. 经历资产库页 (`/experiences`)

**布局**：三栏自适应（桌面端）/ 单栏 + 底部 Sheet（移动端）

```
┌──────────────────────────────────────────────────────────────┐
│ Header: 经历资产库                                    [+新建] │
├──────────┬─────────────────────────────┬─────────────────────┤
│          │                             │                     │
│ 分类导航  │   经历列表（卡片视图）        │   详情/编辑面板      │
│          │                             │   （点击后滑入）     │
│ • 全部   │   ┌─────────────────┐       │                     │
│ • 实习   │   │ ExperienceCard  │       │  [类型切换标签]      │
│ • 项目   │   │ - 标题          │       │  [动态表单]          │
│ • 教育   │   │ - 时间          │       │  [AI 分析区]         │
│ • 技能   │   │ - 标签          │       │                     │
│          │   └─────────────────┘       │                     │
│ 标签云   │                             │                     │
│ [搜索框] │   [加载更多...]              │                     │
│          │                             │                     │
└──────────┴─────────────────────────────┴─────────────────────┘
```

**核心交互**：
- 点击卡片 → 右侧滑出编辑面板
- 新建经历 → 先选类型，再填表单
- 批量选择 → 顶部出现操作浮层

### 2. JD 对齐工作台 (`/jd-workbench`)

**布局**：四阶段步骤条 + 主工作区

```
┌──────────────────────────────────────────────────────────────┐
│  Step 1: 粘贴 JD  →  Step 2: AI 分析  →  Step 3: 经历匹配  →  Step 4: 对齐编辑  │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌────────────────────────────┐  ┌─────────────────────────┐ │
│  │      JD 粘贴/输入区域       │  │    JD 分析结果面板       │ │
│  │      (大文本框)            │  │    - 硬性要求            │ │
│  │                            │  │    - 核心能力            │ │
│  │  [粘贴示例 JD] [开始分析]   │  │    - 加分项              │ │
│  │                            │  │    - 关键词云            │ │
│  └────────────────────────────┘  │    [校正分析结果]        │ │
│                                  └─────────────────────────┘ │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │              经历匹配推荐区（Match Cards）                │ │
│  │  ┌─────────────┐ ┌─────────────┐ ┌─────────────┐       │ │
│  │  │ 匹配度 92%   │ │ 匹配度 78%   │ │ 匹配度 65%   │       │ │
│  │  │ [选中]      │ │ [选中]      │ │ [忽略]      │       │ │
│  │  └─────────────┘ └─────────────┘ └─────────────┘       │ │
│  └─────────────────────────────────────────────────────────┘ │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │              STAR 对齐编辑区（左右对比）                  │ │
│  │  ┌─────────────────────┐  ┌─────────────────────────┐   │ │
│  │  │ 用户原文             │  │ AI 优化建议              │   │ │
│  │  │                     │  │ [置信度: 高]             │   │ │
│  │  │                     │  │                         │   │ │
│  │  │                     │  │ [采纳] [编辑后采纳] [忽略]│   │ │
│  │  └─────────────────────┘  └─────────────────────────┘   │ │
│  │              [生成简历草稿 →]                            │ │
│  └─────────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────┘
```

**核心交互**：
- JD 粘贴后自动触发分析（防抖）
- 经历匹配卡片显示匹配度 + 推荐理由
- STAR 优化为左右对比视图，**必须用户显式采纳**

│  │ [+ 添加区块]      │  │    └─────────────────────────┘    │
│  └──────────────────┘  │                                     │
│                        │    缩放: [50%] [75%] [100%] [fit] │

│  │ [从资产库选择 ▼]   │  │                                     │
│  │                  │  │                                     │
│  │ 内容编辑:         │  │                                     │
│  │ ┌──────────────┐  │  │                                     │

│  │                  │  │                                     │

**核心交互**：
- 左侧拖拽排序区块，点击眼睛图标显隐
- 经历来源下拉框关联资产库
- 简历级微调不影响资产库原文
- 模板切换前自动保存快照
